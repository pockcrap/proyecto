Programación

La programación convierte ideas en páginas, menús y juegos interactivos. Este proyecto muestra lo que se puede construir con HTML, CSS y JavaScript.

  • Interfaces modernas.
  • Navegación dentro de una sola página.
  • Juegos y botones interactivos.

Aprender

Temas que se verán en la materia:

  • Lógica y algoritmos.
  • Variables, condiciones y ciclos.
  • HTML, CSS y JavaScript.
  • Diseño responsive.

Bases

Lo aprendido en el curso:

  • Resolver problemas paso a paso.
  • Crear páginas y organizar contenido.
  • Usar eventos, menús y estilos.
  • Probar, corregir y mejorar el código.

Experiencia

Programar enseña a pensar, probar y corregir. Cada error ayuda a entender mejor cómo funciona el código y cómo mejorar una solución.

Crear

Este proyecto busca motivar a otros estudiantes mostrando que con programación se pueden crear páginas, menús, botones y juegos.
